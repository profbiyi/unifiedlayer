## BUSINESS DETAILS

### 22. Business Model
**Subscription**

> Specifically: SaaS subscription with a usage-based component.
>
> **Pricing structure:**
> - **Starter (Free):** 3 connectors, 10,000 rows/month — acquisition funnel
> - **Professional ($29–49/month in local currency):** Unlimited connectors, 500K rows/month, scheduling, quality checks
> - **Enterprise (Custom):** Unlimited everything, SLA, dedicated support, SSO
>
> **Key differentiator:** Priced in local currencies (NGN, KES, GHS, ZAR, XOF) via local payment gateways (Paystack, Flutterwave). No USD conversion. African SMEs save 5–15% vs. paying for US-priced SaaS tools.

### 23. Solution Maturity Stage
**Full Product**

> Justification — the platform currently includes:
> - 8 production-ready source connectors (PostgreSQL, MySQL, REST API, M-Pesa, WhatsApp Business, S3, GCS, Azure Blob)
> - 8 destination types (PostgreSQL, DuckDB, BigQuery, Snowflake, Redshift, S3, GCS, Azure Blob)
> - Full pipeline orchestration with Prefect (scheduling, retries, exponential backoff)
> - Multi-tenant architecture with 3-tier RBAC (Super Admin, Org Admin, Org User)
> - Data lineage tracking at table and column level with graph visualization
> - 7 data quality check types (row count, null checks, uniqueness, freshness, custom SQL)
> - Auto-scaling worker pool (1–10 workers based on queue depth)
> - Prometheus + Grafana monitoring stack
> - Team invitation system with email-based onboarding
> - Change Data Capture via PostgreSQL logical replication and MySQL binlog
> - Complete Next.js 14 frontend with dashboard, pipeline builder, lineage viewer
> - Kubernetes deployment configs with HPA scaling (3–10 replicas)
>
> This is beyond MVP. This is a production-grade product ready for users.
